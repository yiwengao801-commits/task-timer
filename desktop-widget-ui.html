🖥️ 桌面悬浮组件 UI 设计

📋 功能说明

  • 悬浮计时器：始终置顶显示，可拖动位置，显示当前任务和计时
  • 健康提醒：每 40-60 分钟随机弹出，点击「已完成」才消失
  • 置顶功能：点击 📌 按钮切换是否始终置顶
  • 最小化：点击 — 按钮最小化到标题栏
